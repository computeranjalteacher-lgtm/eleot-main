# دليل تجربة تطبيق ELEOT AI Evaluator

## الخطوة 1: تحميل الإضافة في Chrome

1. افتح متصفح Google Chrome
2. اذهب إلى `chrome://extensions/` في شريط العنوان
3. فعّل "وضع المطور" (Developer mode) من الزاوية اليمنى العلوية
4. انقر على "تحميل بدون تعبئة" (Load unpacked)
5. اختر مجلد المشروع (المجلد الذي يحتوي على `manifest.json`)

✅ يجب أن تظهر الإضافة الآن في شريط أدوات Chrome!

## الخطوة 2: إنشاء أيقونات الإضافة (اختياري)

1. افتح ملف `generate-icons.html` في المتصفح
2. انقر على "Download All Icons" (تحميل جميع الأيقونات)
3. انسخ الملفات الثلاثة (`icon16.png`, `icon48.png`, `icon128.png`) إلى مجلد `icons/`
4. أعد تحميل الإضافة في صفحة `chrome://extensions/`

## الخطوة 3: تجربة الإضافة بدون مفتاح API

الإضافة تعمل مع بيانات تجريبية حتى بدون مفتاح API!

1. انقر على أيقونة الإضافة في شريط الأدوات
2. اختر اللغة: العربية أو الإنجليزية
3. الصق وصف حصة تجريبي (انظر الأمثلة أدناه)
4. انقر على "قيّم" (Evaluate)
5. انتظر قليلاً - ستظهر النتائج تلقائياً

### مثال وصف حصة بالعربية:

```
بدأ المعلم الحصة بسؤال تحفيزي حول عملية البناء الضوئي. عمل الطلاب في مجموعات صغيرة لإجراء تجارب على النباتات ومصادر الضوء. تجول المعلم بين المجموعات، وقدم ملاحظات وطرح أسئلة تحفيزية. قدم الطلاب نتائجهم للصف باستخدام العروض الرقمية. تم التقييم من خلال العروض الجماعية والاختبارات الفردية. دمجت الحصة التكنولوجيا من خلال الأجهزة اللوحية والسبورة التفاعلية.
```

### مثال وصف حصة بالإنجليزية:

```
The teacher began the lesson by asking students to share their prior knowledge about photosynthesis. Students worked in small groups to conduct experiments with plants and light sources. The teacher circulated among groups, providing feedback and asking probing questions. Students presented their findings to the class using digital presentations. Assessment was conducted through both group presentations and individual quizzes. The lesson incorporated technology through tablets and interactive whiteboards.
```

## الخطوة 4: اختبار الميزات

### أ) اختبار تبديل اللغة
- اختر العربية وتحقق من تغيير اتجاه النص (RTL)
- اختر الإنجليزية وتحقق من تغيير الاتجاه (LTR)

### ب) اختبار زر النسخ
- بعد ظهور النتائج، انقر على زر "نسخ" بجانب أي تبرير
- يجب أن يظهر "تم النسخ!" مؤقتاً
- الصق في أي مكان للتحقق من نسخ النص

### ج) اختبار التصدير
- **تصدير CSV**: انقر على "تصدير CSV" - يجب أن يحمل ملف CSV
- **تصدير PDF**: انقر على "تصدير PDF" - يجب أن يحمل ملف PDF
- **تصدير Word**: انقر على "تصدير Word" - يجب أن يحمل ملف Word

### د) اختبار النتيجة الإجمالية
- تحقق من ظهور النتيجة الإجمالية (متوسط جميع الدرجات)
- يجب أن تكون بين 1 و 4

## الخطوة 5: إعداد مفتاح API (اختياري - للاستخدام الفعلي)

إذا أردت استخدام الذكاء الاصطناعي الحقيقي بدلاً من البيانات التجريبية:

### الطريقة 1: عبر Console المتصفح

1. افتح الإضافة (انقر على الأيقونة)
2. اضغط F12 أو انقر بزر الماوس الأيمن → "فحص" (Inspect)
3. اذهب إلى تبويب "Console"
4. الصق هذا الأمر (استبدل `your-api-key` بمفتاحك الفعلي):

```javascript
chrome.storage.local.set({ apiKey: 'your-api-key-here' }, () => {
  console.log('تم حفظ مفتاح API بنجاح!');
});
```

5. أغلق الإضافة وأعد فتحها
6. جرب التقييم مرة أخرى - ستستخدم الآن API الحقيقي

### الطريقة 2: استخدام Proxy Server (أكثر أماناً)

1. افتح Terminal في مجلد المشروع
2. ثبت المكتبات المطلوبة:
   ```bash
   npm install
   ```
3. أنشئ ملف `.env` وأضف:
   ```
   LLM_API_KEY=your-api-key-here
   ```
4. شغّل السيرفر:
   ```bash
   node proxy.js
   ```
5. في Console المتصفح، اضبط Endpoint:
   ```javascript
   chrome.storage.local.set({ 
     apiEndpoint: 'http://localhost:3000/api/chat' 
   });
   ```

## استكشاف الأخطاء

### الإضافة لا تظهر؟
- تأكد من تفعيل "وضع المطور"
- تحقق من وجود ملف `manifest.json` في المجلد
- افحص Console للأخطاء (F12)

### لا تظهر النتائج؟
- افتح Console (F12) وابحث عن أخطاء
- تأكد من وجود ملف `config/eleot_ai_config.json`
- جرب نص تجريبي مختلف

### أخطاء API؟
- تحقق من صحة مفتاح API
- تأكد من أن Endpoint صحيح
- افحص Console للأخطاء التفصيلية

### الأيقونات مفقودة؟
- استخدم `generate-icons.html` لإنشاء الأيقونات
- أو اتركها - الإضافة ستعمل بأيقونات افتراضية

## نصائح للتجربة

1. **ابدأ بدون API**: جرب أولاً مع البيانات التجريبية للتأكد من عمل الواجهة
2. **اختبر كلا اللغتين**: تأكد من عمل العربية والإنجليزية بشكل صحيح
3. **جرب نصوص مختلفة**: استخدم وصف حصة قصير وآخر طويل
4. **اختبر التصدير**: تأكد من عمل جميع أنواع التصدير
5. **تحقق من النسخ**: تأكد من عمل أزرار النسخ بشكل صحيح

## ما يجب أن تراه

بعد النقر على "قيّم"، يجب أن ترى:

1. **مؤشر تحميل** (spinner) أثناء المعالجة
2. **جدول النتائج** يحتوي على:
   - 7 عناصر ELEOT
   - درجة لكل عنصر (1-4)
   - تبرير لكل عنصر
   - اقتراح تحسين لكل عنصر
   - زر نسخ لكل تبرير
3. **النتيجة الإجمالية** في الأعلى
4. **أزرار التصدير** (PDF, CSV, Word)

## اختبار سريع (30 ثانية)

1. افتح الإضافة
2. اختر اللغة
3. الصق أي نص (حتى كلمة واحدة)
4. انقر "قيّم"
5. انتظر النتائج

✅ إذا ظهرت النتائج، فالإضافة تعمل بشكل صحيح!

**ملاحظة**: الإضافة تعمل بشكل كامل حتى بدون مفتاح API باستخدام بيانات تجريبية. استخدم مفتاح API فقط إذا أردت تقييمات حقيقية من الذكاء الاصطناعي.
