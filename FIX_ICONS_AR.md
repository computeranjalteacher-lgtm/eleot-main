# حل مشكلة الأيقونات المفقودة

## المشكلة
تظهر رسالة خطأ: "Could not load icon 'icons/icon16.png'"

## الحل السريع (تم تطبيقه)

تم تعديل `manifest.json` لإزالة الأيقونات مؤقتاً. الإضافة ستعمل الآن بدون أيقونات (ستستخدم أيقونة Chrome الافتراضية).

## الحل الكامل: إنشاء الأيقونات

### الطريقة 1: استخدام الملف HTML (الأسهل)

1. **افتح الملف** `create-icons-simple.html` في متصفح Chrome
2. **انقر على الزر** "إنشاء وتحميل جميع الأيقونات"
3. **ستحمل 3 ملفات** تلقائياً:
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`
4. **انسخ الملفات** إلى مجلد `icons/` في مشروعك
5. **أعد تحميل الإضافة** في `chrome://extensions/`

### الطريقة 2: استخدام generate-icons.html

1. افتح `generate-icons.html` في المتصفح
2. انقر "Download All Icons"
3. انسخ الملفات إلى مجلد `icons/`

### بعد إنشاء الأيقونات

1. **أعد تفعيل الأيقونات** في `manifest.json`:

افتح `manifest.json` وأضف هذا الكود:

```json
"action": {
  "default_popup": "popup.html",
  "default_icon": {
    "16": "icons/icon16.png",
    "48": "icons/icon48.png",
    "128": "icons/icon128.png"
  }
},
"icons": {
  "16": "icons/icon16.png",
  "48": "icons/icon48.png",
  "128": "icons/icon128.png"
},
```

2. **أعد تحميل الإضافة** في Chrome

## ملاحظة

الإضافة تعمل بشكل كامل حتى بدون أيقونات. الأيقونات فقط للتحسين البصري. يمكنك استخدام الإضافة الآن بدون أي مشاكل!
